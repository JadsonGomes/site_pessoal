SITE PESSOAL

Este é meu projeto de site pessoal, o mesmo no qual provávelmente você está lendo este conteúdo, um espaço que uso para por em prática meus conhecimentos no desenvolvimento web. Aqui você encontrará reflexões baseadas nos conhecimentos que tenho adquirido ao longo do caminho.

Espero que meu site possa inspirar você de alguma forma. Sinta-se à vontade para deixar comentários, sugestões ou simplesmente passear pelo conteúdo. Obrigado por fazer parte desta jornada!

BLOG "IMPRESSIONADOR"

Este é o repositório do projeto de blog em andamento, desenvolvido em colaboração com a escola Hasthag Treinamentos. O objetivo deste projeto é criar um blog para a comunidade impressionadora compartilhar seus conhecimentos e aprendizados com a comunidade.

Utilizando a poderosa tecnologia Flask, este projeto de blog é uma trilha de aprendizado completa, incorporando HTML, CSS e diversas bibliotecas Python. Ao navegar pelo repositório, você encontrará códigos, templates e recursos que refletem a jornada de aprendizado em desenvolvimento web.

O projeto está em constante evolução, alinhado com o meu progresso nos estudos. Convido você a explorar este repositório , onde cada commit representa um novo passo na criação desta plataforma de aprendizado.

EXERCÍCIOS EM PYTHON

AQUI você encontrará uma coleção de desafios BACK-END cuidadosamente selecionados para praticar e aprimorar suas habilidades na linguagem Python.

Estes exercícios são uma mistura de problemas clássicos e cenários do mundo real, projetados para abordar diferentes aspectos da programação em Python. Sempre que possível eu adiciono comentários aos exercícios que vou resolvendo, explicando a lógica por trás da solução e fornecendo dicas úteis para quem está aprendendo.

Sinta-se à vontade para explorar, clonar ou contribuir com este repositório. Se você é novo na linguagem Python, esses exercícios podem ser um excelente ponto de partida. Se você já tem experiência, eles podem ser úteis para revisão e prática.

Meu objetivo ao compartilhar esses exercícios é criar um recurso educacional acessível para a comunidade. Espero que você encontre valor e diversão ao trabalhar nesses desafios. Não hesite em deixar comentários de melhorias nos exercícios, sugestões ou novos exercícios que gostaria de ver adicionados!
